HTML TABLE 데이터 변환기

📌 사용 안내

  • 기존 테이블 HTML을 먼저 입력해주세요.
  • 데이터는 HTML 테이블 붙여넣기 또는 엑셀 업로드 중 하나로 입력 가능합니다.
  • 데이터 테이블의 텍스트와 병합 구조(rowspan, colspan)만 반영됩니다.
  • 기존 테이블의 스타일, 정렬, 내부 태그(p, span 등)는 유지됩니다.

⚠️ 주의사항

  • 기존 테이블과 데이터 테이블의 열 개수가 크게 다르면 결과가 깨질 수 있습니다.
  • 엑셀 서식(색상, 폰트 등)은 반영되지 않습니다.
  • 복잡한 병합 구조는 일부 수동 수정이 필요할 수 있습니다.

❗ 오류 발생 시

  • 테이블 태그(<table>)가 정상적으로 포함되어 있는지 확인해주세요.
  • 데이터 테이블이 비어있지 않은지 확인해주세요.
  • 엑셀 업로드 시 파일 형식(.xlsx, .xls, .csv)을 확인해주세요.

📧 문의

사용 중 문제가 발생하거나 개선 요청이 있으시면 아래 이메일로 문의해주세요.

1) 기존 테이블 HTML

기존 테이블의 style / class / valign / p / span / colgroup 등을 최대한 유지합니다.

2) 데이터 입력

데이터 테이블에서는 구조(tr/td/th, rowspan, colspan)와 텍스트만 사용합니다.
첫 번째 행은 헤더로 처리합니다. 병합 셀이 있으면 가능한 범위에서 rowspan, colspan을 반영합니다.
분석 결과가 여기에 표시됩니다.
